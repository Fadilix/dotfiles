# how to setup
### Clone the repo
```bash
git clone https://github.com/Fadilix/dotfiles
```
### Go to folder
```bash
cd dotfiles
```

### Give executable right to the files
```bash
chmod +x install.sh
chmod +x backup.sh
```

### Launch the installation
./install.sh

### Backup all your data
./backup.sh
